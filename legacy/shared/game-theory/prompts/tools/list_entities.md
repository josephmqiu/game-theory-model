List all entities of a given type with their IDs and key fields, providing an overview of what has been recorded so far.
Input: `type` (required — entity type such as player, game, source, observation, claim, inference, assumption, scenario).
Use this to find entity IDs for subsequent updates, to check coverage before phase checkpoints, or to identify gaps in the analysis.
For a complete view of a specific entity once you have its ID, use get_entity rather than this tool.
