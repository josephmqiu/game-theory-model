Create a scenario with a narrative, probability estimate, and link to the formal model that drives the prediction.
Inputs: `name` (required), `narrative` (required — what happens and why, stated specifically), `probability_value` (0–1), `formalization_id` (which formalization or game drives this scenario's equilibrium prediction).
Scenarios should sum to approximately 100%; if they don't, there is either a missing scenario or miscalibrated probabilities.
Distinguish equilibrium-based predictions (driven by the formal model) from discretionary forecasts (driven by judgment) — the source of each probability estimate matters for how much weight to place on it.
