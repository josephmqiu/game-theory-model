Returns the detailed analytical methodology for a specific phase, including purpose, sub-steps, and relevant tools.
Input: `phase` (required number 1–10).
Call this when entering a new phase to refresh on the detailed guidance, specific analytical questions to address, and the quality bar for that phase's outputs.
Do not rely on memory of phase guidance from earlier in a long session — call this tool at the start of each phase to ensure the analysis follows the current methodology.
