Make an assumption embedded in the analysis explicit and rated, so it can be tracked and tested.
Inputs: `statement` (required — what you're assuming), `type` (behavioral/capability/structural/institutional/rationality/information), `sensitivity` (critical/high/medium/low — critical means flipping this assumption flips the central prediction), `evidence_quality` (direct_evidence/inference/assumption_only).
Every analysis contains assumptions; making them explicit is how you build a falsifiable model rather than a narrative that cannot be evaluated.
Assumptions rated critical or high sensitivity require the strongest supporting evidence — if you cannot ground them, flag them as assumption_only so readers can stress-test them.
